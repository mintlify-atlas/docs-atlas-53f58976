description: "Open-source reimplementation of the MechCommander 2 RTS engine with OpenGL rendering, cross-platform Linux and Windows support, and a full asset pipeline."
---

MechCommander 2 Open Source Engine is a community-maintained reimplementation of the original MC2 engine, built from the game's published header files. It runs the full single-player campaign on both Linux and Windows in 64-bit mode, with OpenGL rendering replacing the original DirectX pipeline and SDL2 handling audio, input, and windowing.
